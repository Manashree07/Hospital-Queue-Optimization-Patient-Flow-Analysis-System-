# Hospital-Queue-Optimization-Patient-Flow-Analysis-System-

##  Project Overview

This project focuses on reducing patient waiting time using a queue optimization system. It uses Python for processing, MySQL for data storage, and Power BI for visualization.

##  Features

* Priority-based queue system
* Optimized waiting time calculation
* Doctor-wise scheduling
* Interactive Power BI dashboard

##  Technologies Used

* Python (Pandas)
* MySQL
* Power BI

##  Output

* Optimized dataset (CSV)
* Power BI dashboard

##  Project Structure

* code/ → Python logic
* data/ → Dataset files
* dashboard/ → Power BI file
